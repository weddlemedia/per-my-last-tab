# Per My Last Tab

A Chrome extension that replaces your New Tab page with dry, satirical corporate wisdom.

Every new tab is a fresh reminder that synergy is just a word we use when no one knows whose job it is.

## Features

- 60 handcrafted quotes dripping with corporate satire
- Absurd fake job titles for every attribution
- Rotating muted color palette (boardroom-approved)
- Zero permissions, zero tracking, zero data collection
- Loads instantly — no network requests, no nonsense
- Manifest V3 compliant

## Install (Developer Mode)

1. Clone this repo:
   ```
   git clone https://github.com/weddlemedia/per-my-last-tab.git
   ```
2. Open Chrome and navigate to `chrome://extensions`
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked**
5. Select the `per-my-last-tab` folder
6. Open a new tab and enjoy your daily dose of corporate wisdom

## Privacy

This extension collects no data. None. It doesn't even know you're here. Everything runs locally in your browser.
